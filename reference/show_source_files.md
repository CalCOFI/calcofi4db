# Show source files

Show source CSV files with local and GCS paths for provenance

## Usage

``` r
show_source_files(d)
```

## Arguments

- d:

  data object output from
  [`read_csv_files()`](https://calcofi.io/calcofi4db/reference/read_csv_files.md)

## Value

DT datatable showing source files with paths and row counts
