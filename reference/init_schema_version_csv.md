# Initialize Schema Version CSV

Creates an empty schema_version.csv file in inst/ directory if it
doesn't exist. This is useful for starting a new versioning system.

## Usage

``` r
init_schema_version_csv()
```

## Value

Invisible NULL (CSV file is created)

## Examples

``` r
if (FALSE) { # \dontrun{
init_schema_version_csv()
} # }
```
